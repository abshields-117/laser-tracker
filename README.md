# Harlan Laser Tracker (MVP)

## Getting Started

This is a Next.js application built for tracking Laser Hair Removal sessions.

### 1. Install Dependencies
```bash
npm install
```

### 2. Run Locally
```bash
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) with your browser.

### 3. Deployment (Beta)
To deploy this for staff use:
1.  Push this folder to a private GitHub repo.
2.  Connect the repo to **Vercel** (free account).
3.  It will build automatically.

## Features Included
*   **Patient Search:** Filter by name/DOB.
*   **Session Progress:** Visual "Session 4/8" bars.
*   **Treatment Log:** Fields for Fluence, Pulse Width, Spot Size (Splendor X specific).
*   **Skin Reaction:** Quick checkboxes for Erythema/Edema.

## Tech Stack
*   **Framework:** Next.js 14
*   **Styling:** Tailwind CSS
*   **Icons:** Lucide React
